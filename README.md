# 🎬 CineSuggestion

Netflix-inspired intelligent movie recommendation system developed using Python Flask and SQLite.

## Features
- User Registration and Login
- Session Management
- Movie Search
- Content-Based Recommendations
- Personal Watchlist
- Persistent Database Storage
- Netflix-Inspired UI

## Technologies Used
- Python
- Flask
- HTML
- CSS
- SQLite
- Git & GitHub

## Project Structure

CineSuggestion/
├── app.py
├── database.py
├── create_tables.py
├── movies.py
├── templates/
├── static/
└── database.db

## Run Project

pip install flask
python create_tables.py
python app.py
